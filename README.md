# 👁 Eye Focus Trainer

Eye Focus Trainer is a simple **Progressive Web App (PWA)** with exercises designed to reduce eye strain and train visual focus.

The app works offline and can be installed on mobile devices like a native application.

---

## ✨ Features

- 🎯 **Focus Training**  
  Moving target exercise to train the eye's focusing ability.

- ⏱ **20-20-20 Timer**  
  Reminds you to look away every 20 minutes to relax your eyes.

- 👁 **Blink Exercise**  
  Guided blinking exercise to help moisturize the eyes.

- 🔠 **Visual Acuity Test**  
  Simple eye chart test with multiple difficulty levels.

- 📊 **Training Statistics**  
  Tracks sessions and progress.

- 📱 **Installable PWA**  
  Works offline and can be added to the home screen.

3. The app will work offline
